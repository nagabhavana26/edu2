## How It Works - Step by Step

### Architecture Flow

```
User says: "What is machine learning?"
    ↓
┌─────────────────────────────────────┐
│ 1. VOICE/TEXT INPUT                 │
│ - Mic button captures audio         │
│ - Or user types text                │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 2. FRONTEND PROCESSING              │
│ - React component manages state     │
│ - Displays message in UI            │
│ - Shows typing indicator            │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 3. HTTP REQUEST TO BACKEND          │
│ POST /api/chat                      │
│ {                                   │
│   message: "What is machine...",    │
│   userId: "user_123"                │
│ }                                   │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 4. BACKEND PROCESSING               │
│ - Save user message to history      │
│ - Build context from previous msgs  │
│ - Add system prompt                 │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 5. OPENAI API CALL                  │
│ Send conversation to GPT-3.5        │
│ with full context                   │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 6. AI RESPONSE                      │
│ "Machine learning is a subset..."   │
│ (Generated by OpenAI)               │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 7. SAVE TO DATABASE                 │
│ Store in chat_history table         │
│ (Supabase PostgreSQL)               │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 8. RETURN TO FRONTEND               │
│ {                                   │
│   message: "Machine learning...",   │
│   success: true                     │
│ }                                   │
└─────────────────────────────────────┘
    ↓
┌─────────────────────────────────────┐
│ 9. DISPLAY & OUTPUT                 │
│ - Show AI response in chat          │
│ - Optional: Speak response          │
│ - Ready for next question           │
└─────────────────────────────────────┘
```

## API Endpoints Available

### 1. Chat Endpoint
**POST** `/api/chat`
- Send messages and receive AI responses
- Maintains conversation context
- Stores automatically

### 2. Health Check
**GET** `/api/health`
- Verify server is running
- Check API status

### 3. Chat History
**GET** `/api/chat-history/:userId`
- Retrieve all previous conversations
- Sorted by date

### 4. Clear Chat
**POST** `/api/chat-clear/:userId`
- Start fresh conversation
- Clear memory

## Database Schema

### chat_history Table
```sql
CREATE TABLE chat_history (
  id BIGINT PRIMARY KEY,
  user_id TEXT,                    -- Which user (random ID if not logged in)
  user_message TEXT,               -- What user asked
  assistant_message TEXT,          -- AI response
  created_at TIMESTAMP,            -- When created
  updated_at TIMESTAMP             -- When updated
);
```

### ai_context Table
```sql
CREATE TABLE ai_context (
  id BIGINT PRIMARY KEY,
  user_id TEXT UNIQUE,             -- User identifier
  preferred_language TEXT,         -- en, es, fr, etc.
  learning_goals TEXT,             -- What user wants to learn
  skill_level TEXT,                -- Beginner, Intermediate, Advanced
  interests TEXT[],                -- Array of topics
  conversation_summary TEXT,       -- Summary of learnings
  created_at TIMESTAMP,
  updated_at TIMESTAMP
);
```

## Getting Started - Quick Reference

### 1️⃣ Get API Keys (5 mins)

**OpenAI:**
→ Visit: https://platform.openai.com/api-keys
→ Create new secret key
→ Copy value starting with `sk-`

**Supabase:**
→ Visit: https://app.supabase.com
→ Select your project
→ Settings → API
→ Copy Project URL and create service role key

### 2️⃣ Configure Backend (5 mins)

```bash
cd server
cp .env.example .env
# Edit .env with your keys
npm install
```

### 3️⃣ Setup Database (2 mins)

1. Supabase Dashboard → SQL Editor
2. Create new query
3. Copy SQL from: `supabase/migrations/20260225_ai_assistant_schema.sql`
4. Execute

### 4️⃣ Start Everything (2 mins)

**Terminal 1:**
```bash
cd server && npm run dev
# Port 5000
```

**Terminal 2:**
```bash
npm run dev
# Port 8080
```

### 5️⃣ Test (1 min)

1. Open http://localhost:8080
2. Dashboard → AI Assistant
3. Type or speak a question
4. Get instant AI response!

```javascript
const systemMessage = `You are EduFlow AI...
```

In `server/index.js`:
```javascript
model: 'gpt-3.5-turbo'  // Or 'gpt-4', 'gpt-3.5-turbo-16k'
```

### Adjust Response Style
Modify temperature and max tokens:
```javascript
temperature: 0.7    // 0=precise, 1=creative
max_tokens: 1000    // How long response can be
```

- Change colors
